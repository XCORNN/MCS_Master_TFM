Puedes ejecutar las aplicaciones desde el menú de aplicaciones o directamente desde la carpeta de instalación.

THEHARVESTER
Para usar theHarvester:
 1. Navega al directorio del programa:
     cd ~/Escritorio/theHarvester/theHarvester
 2. Activa el entorno virtual:
     source venv/bin/activate
 3. Ejecuta el programa:
     python3 theHarvester.py

TOR SERVICE
Para usar TOR SERVICE:
 1. Ejecuta en una terminal para iniciar el servicio:
    systemctl start tor
 2. Habilita el servicio para que inicie al arrancar el sistema:
    systemctl enable tor

RECON-NG

 3. Ejecuta el programa:

FOCA
Para usar FOCA:
 1. Navega al directorio del programa:
    cd ~/Escritorio/FOCA
 2. Ejecuta el programa con Wine:
    wine Foca.exe
 3. Conéctate usando los siguientes datos:
    - Host: localhost
    - Usuario: SA
    - Contraseña: Passw0rd!

FOCA_SQL_EXPRESS
Para usar FOCA_SQL_EXPRESS:
 1. Ejecuta en una terminal:
    /opt/mssql-tools/bin/sqlcmd -S localhost -U SA -P 'Passw0rd!'
